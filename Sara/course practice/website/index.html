Link Targets

Link Targets

You can nominate whether to open the URL in a new window or the current window using the target attribute:

_blank

Opens the URL in a new browser window.

Google

_self

Loads the URL in the current browser window.

_parent

Loads the URL into the parent frame (still within the current browser window).

_top

Loads the URL in the current browser window, but cancels out any frames.
